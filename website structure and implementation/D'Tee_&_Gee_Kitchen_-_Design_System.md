# D'Tee & Gee Kitchen - Design System

## Brand Analysis

Based on the logo analysis, D'Tee & Gee Kitchen represents a warm, friendly, and professional food brand that specializes in natural flavored drinks, smoothies, parfaits, zobo drinks, yogurt, chin chin, and small chops.

### Logo Elements
- **Character**: Friendly female chef with warm smile
- **Chef Hat**: Light blue/teal color suggesting freshness and cleanliness
- **Apron**: Red color indicating passion and appetite appeal
- **Serving Tray**: Professional service orientation
- **Wheat Decorations**: Natural, organic, wholesome ingredients
- **Typography**: Gold/yellow elegant serif font

## Color Palette

### Primary Colors
- **Brand Gold**: #D4AF37 (main brand color from logo text)
- **Chef Blue**: #7FB3D3 (from chef hat - freshness, trust)
- **Appetite Red**: #C41E3A (from apron - passion, energy)
- **Wheat Gold**: #F4D03F (from wheat decorations - natural, organic)

### Secondary Colors
- **Warm Brown**: #8B4513 (from chef's hair - earthiness, comfort)
- **Cream White**: #FFF8DC (background, cleanliness)
- **Soft Mint**: #98FB98 (natural, healthy)
- **Deep Teal**: #008B8B (professional, fresh)

### Dark Mode Palette
- **Dark Background**: #1A1A1A
- **Dark Surface**: #2D2D2D
- **Dark Gold**: #B8860B
- **Muted Blue**: #4682B4
- **Soft Red**: #CD5C5C

## Typography

### Primary Font Stack
- **Headings**: 'Playfair Display', serif (elegant, sophisticated)
- **Body Text**: 'Inter', sans-serif (modern, readable)
- **Accent Text**: 'Dancing Script', cursive (friendly, personal)

### Font Sizes (Responsive)
- **H1**: 3.5rem (mobile: 2.5rem)
- **H2**: 2.5rem (mobile: 2rem)
- **H3**: 2rem (mobile: 1.5rem)
- **Body**: 1.1rem (mobile: 1rem)
- **Small**: 0.9rem

## Design Principles

### Visual Style
- **Warm & Inviting**: Use warm colors and friendly imagery
- **Professional**: Clean layouts with proper spacing
- **Natural**: Organic shapes and natural imagery
- **Modern**: Contemporary web design with smooth animations
- **Accessible**: High contrast ratios and readable fonts

### Animation Style
- **Smooth Transitions**: 0.3s ease-in-out for most interactions
- **Bounce Effects**: Subtle spring animations for buttons
- **Fade Animations**: Smooth content reveals on scroll
- **Hover States**: Gentle scale and color transitions
- **Loading States**: Elegant skeleton screens and spinners

## UI Components Style Guide

### Buttons
- **Primary**: Gold gradient with white text, rounded corners
- **Secondary**: Transparent with gold border
- **Hover**: Subtle scale (1.05) and shadow effects

### Cards
- **Background**: White with subtle shadow
- **Border Radius**: 12px
- **Hover**: Lift effect with increased shadow

### Forms
- **Input Fields**: Rounded borders with focus states
- **Labels**: Floating labels with smooth transitions
- **Validation**: Inline feedback with color coding

### Navigation
- **Header**: Sticky navigation with blur background
- **Mobile**: Hamburger menu with slide-in animation
- **Active States**: Gold underline or background highlight

## Layout System

### Grid System
- **Desktop**: 12-column grid with 24px gutters
- **Tablet**: 8-column grid with 20px gutters
- **Mobile**: 4-column grid with 16px gutters

### Breakpoints
- **Mobile**: 320px - 768px
- **Tablet**: 768px - 1024px
- **Desktop**: 1024px - 1440px
- **Large**: 1440px+

### Spacing Scale
- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **2xl**: 48px
- **3xl**: 64px

## Content Strategy

### Hero Section
- **Video Background**: Appetizing food preparation videos
- **Overlay**: Semi-transparent with brand messaging
- **CTA**: Prominent "Order Now" button

### Product Sections
- **Categories**: Drinks, Smoothies, Parfaits, Snacks
- **Product Cards**: High-quality images with hover effects
- **Quick Add**: One-click add to cart functionality

### Features
- **User Accounts**: Registration and login system
- **Shopping Cart**: Persistent cart with quantity controls
- **Payment Integration**: Secure payment processing
- **Reviews**: Customer feedback and ratings
- **Order Tracking**: Real-time order status updates

## Technical Specifications

### Frontend Stack
- **HTML5**: Semantic markup
- **CSS3**: Custom properties, Grid, Flexbox
- **JavaScript**: ES6+ with modern features
- **Animations**: CSS transitions and keyframes

### Backend Stack
- **Flask**: Python web framework
- **SQLite**: Database for development
- **RESTful API**: JSON-based communication
- **Authentication**: Session-based with password hashing

### Performance Goals
- **Page Load**: < 3 seconds
- **First Contentful Paint**: < 1.5 seconds
- **Lighthouse Score**: > 90
- **Mobile Optimization**: Touch-friendly interactions

## Accessibility Standards

### WCAG 2.1 Compliance
- **Color Contrast**: Minimum 4.5:1 ratio
- **Keyboard Navigation**: Full keyboard accessibility
- **Screen Readers**: Proper ARIA labels and roles
- **Focus Indicators**: Visible focus states
- **Alternative Text**: Descriptive alt text for images
